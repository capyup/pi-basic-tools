## Install

The npm package is now `@capyup/capy-tools`:

```bash
pi install npm:@capyup/capy-tools
pi update npm:@capyup/capy-tools
```

If you still have the old npm name installed, remove it after upgrading:

```bash
pi remove npm:@capyup/pi-basic-tools
```
